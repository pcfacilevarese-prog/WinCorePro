Funzionalità

Tutto ciò che serve
per diagnosticare un PC.

WinCurePro copre ogni aspetto dell'ispezione professionale di un PC Windows.

🔬
Diagnostica Hardware

CPU, RAM, dischi, temperature e seriali via WMI e psutil. Soglie configurabili. CPU warn >80%, critico >95%.

📊
Punteggio Salute 0–100

Algoritmo ponderato con deduzioni per ogni anomalia. Verdetto: BUONO ≥75 · ATTENZIONE ≥50 · CRITICO <50.

🔋
Analisi Batteria

Salute, cicli, chimica e autonomia stimata. Alert automatici sotto 20% carica, warning sotto 75% salute.

📄
Report Professionali

PDF con ReportLab, HTML interattivo, CSV, TXT. Firma SHA-256 per autenticità del documento.

🔐
Cifratura + 2FA

AES-128 Fernet + TOTP 2FA. Timeout sessione 4 ore. Max 5 tentativi login con blocco 30s.

👤
Gestione Clienti

Database SQLite locale, storico interventi, certificati PDF firmati e profilo tecnico personalizzabile.

🤖
Assistente AI (Piano Business)

Chat diagnostica in linguaggio naturale. Chiedi "perché il PC è lento?" e ricevi un'analisi immediata basata sui dati reali del sistema.

🔮
AI Predittiva (Piano Business)

Analisi delle tendenze hardware in tempo reale. Regressione lineare su CPU, RAM e temperatura: rileva anomalie prima che diventino guasti.

Nessuna responsabilità per danni

COME FUNZIONA — WinCurePro legge i dati hardware esclusivamente tramite API Windows standard (WMI, psutil, subprocess di sistema). Non modifica file di sistema, impostazioni di sicurezza, registro di Windows o driver. Opera in sola lettura.

RISULTATI DIAGNOSTICI — I punteggi, le previsioni e i report generati hanno valore informativo e orientativo. Non costituiscono una perizia tecnica certificata, una garanzia sullo stato dell'hardware né un consiglio professionale vincolante. Per decisioni critiche (sostituzione componenti, garanzie contrattuali) affidarsi sempre a un'ispezione fisica da parte di un tecnico qualificato.

FUNZIONI AI — L'Assistente AI (Piano Business) utilizza il servizio Groq API per elaborare le domande. Le query inviate contengono solo dati tecnici anonimi del PC (temperature, carichi, versioni driver) — nessun dato personale, nessun file, nessun documento. L'AI Predittiva opera interamente offline tramite regressione lineare locale senza inviare dati a server esterni.

PRIVACY — Nessun dato personale, di navigazione o di utilizzo viene mai trasmesso a terze parti. La licenza è legata alla chiavetta USB fisica: nessuna telemetria, nessun account cloud obbligatorio.
